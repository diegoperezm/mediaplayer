


# Create a virtual environment


## Macos 15

 - Install Python3.13


```sh
brew install python@3.13
```

 - Create a virtual environment
```sh
python3.13 -m venv .env
```

 - Activate the  environment
```sh
source .env/bin/activate
```

 - Install dependencies

```sh
python -m pip install -e .
```
